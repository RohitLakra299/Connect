
# Connect

A social media media application built with firebase.
<br/>
Get application [Link](https://drive.google.com/file/d/1w8HuipPnE_uw5evygCVbhPq6aQLac1W5/view?usp=sharing)






## Tech Stack

- Kotlin
- Coroutine
- Firestore
- Firebase Authentication
- Firebase Storage


## Features

 - A user can authenticate themselves using email and password, or directly by gmail.
 - User can follow and search other users.
 - A user can post images.
 - User can see feed to see there friends posts.
 - User can create and update there own profile.
 - User can chat with there friends.
